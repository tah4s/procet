# Contributing

Teşekkürler! Bu projeye katkıda bulunmak istiyorsanız lütfen kısa rehberi izleyin.

- Hata bildirmek için Issues kullanın. Başlık ve yeniden üretme adımlarını ekleyin.
- Yeni özellikler veya düzeltmeler için Pull Request açın.
- PR'ler küçük ve odaklı olsun; tek bir sorunu çözen PR'lar tercih edilir.
- Commit mesajları kısa ve açıklayıcı olsun. Örnek: `fix: correct typo in README` veya `feat: add new command`.
- Branch adı: `feature/xxx`, `fix/xxx`, `chore/xxx` gibi yapı kullanın.
- Kod formatı: Python için PEP8 uyumlu olmaya çalışın.
- Testler: Yeni özelliklerle ilgili test ekleyin (varsa `tests/` dizinine).

Teşekkürler! Herhangi bir sorunuz olursa Issue açın veya maintainer'a ulaşın.
